000 朝晖新生出行

浙江工业大学朝晖校区地处市中心——拱墅区，平时出行、假期回家出游都非常方便。

    200 学在工大

    201 学分制

    学校采取学分制，以学分来评估学习量，以绩点来衡量学习质量，以平均绩点（也就是大家应该听到过的GPA）来衡量学习过程总体情况。
    无论是公选课、限选课或者必修课，只要学习该课程并且通过该课程的考试，都可以取得学分 。只有按要求修满本专业规定的学分才可以毕业获得学士学位证书。

    202 关于辅修专业

    辅修专业适合一些学有余力的人，学校所有专业均开设辅修专业，在达到主修专业的毕业条件并修满辅修专业的学分后可以得到辅修专业证书。

    203 关于转专业

    工大的转专业政策比较宽松，如果大家发现自己真的对现在所学的专业不感兴趣/不满意的话可以考虑转专业 。
    【普通生转专业】
    学生(中外合作办学专业学生除外)：在第二、三、四学期，可在全校范围内申请转专业。
    中外合作办学专业学生：第一学年平均学分绩点列专业前2%且无不及格课程，在第二学年初可在全校范围内申请转专业。
    【特长生转专业】
    学生在某一专业上有明显专长，并已取得相应成果，且课程平均绩点在2.0(含)以上，可在第二、三、四学期申请转专业，申请专业应为其特长专业。

注：根据每年各专业转专业细则不同可能会略有出入。

204 关于奖学金

来了大学之后大家肯定都希望自己能够拿到那份传说中的奖学金，其实拿奖学金并没有你想像中的那么难。
工大能拿奖学金的人还是很多的，只要大家努力学习，多多参加学校举办的活动力争优秀，下一个拿奖学金的人就是你哦。！

工大的校设奖学金主要分为优秀学生奖学金和单项奖学金
【优秀学生奖学金】

  1. 一等优秀学生奖学金：要求学生获得一等学习奖学金或者二等学习奖学金和除学习奖学金外的一项单项奖学金，奖金为5000元/人·年
  2. 二等优秀学生奖学金：要求学生获得二等学习奖学金或者三等学习奖学金和除学习奖学金外的一项单项奖学金，奖金为2500元/人·年
    【单项奖学金：】
  3. 一等学习奖学金：获奖比例，<=3%,金额为2000元/人·年
  4. 二等学习奖学金：获奖比例，<=10%,金额为1500元/人·年
  5. 三等奖学习学金：获奖比例，<=20%,金额为500元/人·年

除了以上的学习奖学金外，学校还设立了创新创业奖、社会实践奖、社会工作奖、问题拓展奖四项奖学金来奖励在某方面有突出成绩的学生。四个单项奖学金各项获奖比例<=1%，金额为1500元/人·年。
除了校设奖学金外，还有国家奖学金、省政府奖学金和社会奖学金等其它奖学金等着大家争取，具体评定方案由将颁奖单位制定。

205 关于勤工

来了大学之后就很多人都觉得自己应该自立，自己养活自己。
开学初寝室里会出现各种来历不明的兼职推销人员，精小弘提醒大家谨慎选择！一方面屏峰和莫干山校区地处偏僻，而大多数的兼职都在市区，坐2个小时公交车或花几十块钱打车去一个陌生的地方做兼职并不值得；另一方面各种兼职信息错综复杂，有一部分兼职推销在你提交个人信息甚至付钱之后就消失不见了。
精小弘建议大家选择学校勤工网提供的兼职信息。
勤工报名流程如下：进入浙工大学校官网首页->点击下方导航的“学生社区”->进入“我去勤工网” ->选择学生入口->激活你的账号（用户名和密码都为学号）->完善个人信息->选择适合自己的勤工信息。

206 关于助学贷款

对于一些经济上有困难的学生，学校会提供助学贷款。
原则上每个申请贷款的学生每学年最多申请8000元贷款年度学费和住宿费标准总和低于8000元的，贷款额度可按照学费和住宿标准总和确定。在校期间，助学贷款的利息全部由国家财政补贴，毕业后的贷款利息由贷款学生自付。
申请贷款的过程比较复杂，详情请咨询随录取通知书一同寄来的通知单。这里有几个注意点希望大家注意一下：

  1. 申请助学贷款者需学习认真、品德优良，不能出现挂科的情况，申请时需要提供上一年的成绩单，新生则需要提供高考成绩单。
  2. 不能有刑事犯罪记录、学校严重警告以上的处分记录与不良信用记录。
  3. 申请人家庭确实有困难，有乡及乡以上民政部门提供的家庭经济贫困证明原件。

    207 关于选课

    对于习惯了中小学课程（学校统一安排课表且科目有连续性）的大一新生来说，大学的课程多得让我们眼花缭乱，课程分类也比较复杂，因此选课对我们而言还是一件新鲜而陌生的事物。
    大学的学习与选课有莫大的关系，一个好的老师一门好的选修课，能够激发我们的学习积极性，发现自身的兴趣点，更有可能影响自己的未来走向。所以精小弘这里整理的一些关于选课的资料供大家参考。
    【 体育课】
    大学里的体育课就是大家释放自己天性的时候了。大学体育课除了篮球、足球、排球、网球等常见运动外，还有野外拓展、定向运动、瑜伽等。一般体育课是不会出现挂科情况的，大家可以根据自己的兴趣点自行选择。
    但是如果体测某项没有达到分数标准的话，那学期的体育总成绩不会高于65分哦！

300 新生开学注册流程

到达学校后第一件事情当然就是新生注册！
报道当天，学校一般会在存中楼前/支干路将会按学院设立新生报到处，会有志愿者带你找到自己的学院，然后就可以登记注册，领取相关物品，如果开学前没有交学费，则到支干路学院处报道了以后，先去交款，然后再到寝室去。届时会有志愿者带领你到寝室，在楼下注册登记并领取钥匙后就可在寝室住下了，如果有订购学校的床上用品将会有一个牛津包放在你的床上，除了牙刷牙膏等等小物品需要另外购买，其他的一般都齐备。

在寝室安顿好之后，需要完成以下事情

  1. 体检 ：一般会设立在校医院（朝晖）/养贤府食堂二楼（屏峰），主要项目是身高体重和视力，以及胸透等等，到时候不要忘记带上注册时拿到的体检表格，需要上交的。
  2. 领书：注册时会拿到一张领书单，上面写明了需要领取的书本，到指定的教材发放处领取就可以了。
  3. 缴费：学费一般都是开学前在银行/支付宝缴纳了，不过届时可能还会有别的费用，比如空调、热水器的租金和押金等。另外申请助学贷款的，请在报到处咨询相关事宜。
  4. 其他：比如购物等等，学校里面有超市，大部分生活用品都有的，不过到时候人肯定会很多，所以最好选择好时间，比如晚上或者午饭时间。超市接受现金和刷卡。

    400 住宿问题

    新生入学时，住宿的需求比较大，可能会出现房源紧张的情况，建议避开高峰。外地的新生可以选择住的远一些，比如提前预定好西湖附近的酒店，顺便和家人游一游西湖。

604 精弘毅行是什么活动？

精弘毅行，是浙江工业大学精弘网络参照“毅行者”发起并组织的一项民间远足活动，始于2012年，从2016年起，毅行改为在每年秋天举办，并有志愿者组成工作队伍。一路蜿蜒崎岖，一路山水环抱，一路互相扶持，一路浩浩荡荡，为考验耐力与毅力的良机。精弘毅行在线路的选择上充分发挥地处小和山的优势，屏峰山、小和山等平时非常熟悉的山峰以及国家森林公园午潮山都包含在线路中，让参与者在勇攀高峰的同时一览校园美景。从2018年起，毅行额外增加了朝晖路线，参与毅行的各位可以在古运河河畔感受人文气息，参与轻松愉快的小游戏。正所谓：不走一次精弘毅行，怎么好意思说你来过工大！
