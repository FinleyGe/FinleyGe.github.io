ubuntu软件推荐

发表于 2022-04-12 更新于 2022-04-16 分类于 linux ， 软件推荐

前言

在上一篇 blog 中我曾说要写一篇新的 blog 以推荐我在 ubuntu 下使用的各种软件。

已经鸽了这么长时间，积累了不少好用的软件，推荐给各位。

基础软件

装机后必装的软件。

请在安装软件以前保证您的设备的网络畅通无阻。
由于众所知周的原因，可以参考我的github page
并且更新了您的软件源。（可以参考上一篇文章）

Git

安装

sudo apt install git

配置

git config --global user.name username
git config --global user.email yourEmail@example.com

Google Chrome

Firefox 和 Chrome 的争端是一直以来津津乐道的话题。
笔者更倾向于使用Chrome。
google chrome 官网中提供了.deb格式的安装包。下载后通过sudo dpkg -i filename.deb或者ubuntu自带的软件管理器安装。

Fcitx5

Ubuntu默认使用ibus作为输入法管理器，但是年代久远，不好用。
使用Fcitx5作为新的输入法管理器。

  1. sudo apt install fcitx5-im 安装本体、以及常用的配置等。
  2. sudo apt install fcitx5-chinese-addon 安装中文输入法。
  3. 修改默认输入法
    1. 打开语言支持
    2. 修改fcitx5为默认的输入法
  4. 删除ibus : sudo apt purge ibus

包（软件）管理器

Snappy

也就是snap， 用于ubuntu的包管理器。

sudo apt install snapd

Flatpak

其本质是一个应用程序沙盒。但在实际使用中可以被视为一个包管理器。

AppImageLauncher

AppImageLauncher 是用以管理 AppImage 的一个管理器。
可以在 Github 上下载。

Ubuntu Gnome 桌面美化

笔者曾经使用 Kubuntu 的 KDE Plasma 桌面。不可否认的是 KDE 桌面更为华丽。

但是由于笔者的某些错误操作，导致 Kubuntu 删除了自己的桌面造成了……所以还是选择了Ubuntu

作为一个开发者，或许过于华丽并非是必要的。

但是 美观即是生产力， 对桌面的美化也是需要的，所以推荐以下几个软件。

  • variety
  • Gnome-Tweaks
  • Gnome 插件

Variety

Variety 是一个强大的桌面背景管理器。可以自动从图片源下载图片作为桌面背景。

安装

sudo apt install variety

配置

此软件是开箱即用的，并且支持中文。

Gnome-Tweaks

执行sudo apt install gnome-tweaks安装。
可以修改 gnome 桌面的主题、图表、鼠标等等。

Gnome 插件

在 Chrome 或 Firefox 中可以下载 GNOME Shell integration 拓展，更方便的管理 Gnome

访问：
https://extensions.gnome.org/
以下载gnome插件。

插件推荐

  1. Clipboard Indicator 用于管理剪切板历史
  2. Dash to Dock 替换原有的丑陋的dock栏（不过说实话还是很丑）
  3. Disconnect WiFi 可以断开WiFi链接
  4. GSConnect 是KDE Connect的 gnome 实现，神器，在局域网中和安卓手机联动。安卓手机需要下载KDE Connect
  5. Input Method Panel 可以使fcitx5的KDE界面在Gnome下运行。
  6. Open Weather 天气预报
  7. Recent Items 最近打开的目录、文件
  8. Removable Drive Menu 管理可移除设备（U盘、移动硬盘等）
  9. Sound Input & Output Device Chooser 管理声音输入输出设备

Visual Studio Code

不作介绍，下就完了。

环境配置

个人认为 Linux 下配置环境比 windows 简单。
此处只写几个常用的环境。

  1. cpp
    直接sudo apt install gcc
  2. python
    默认有python
    tips: 默认是 python3, 开启 shell 需要执行 python3

Neovim

sudo snap install neovim

Obsidian

一个支持 双向链接， 支持 Markdown 的编辑器。
很好用。
我将再写一个新的 blog 介绍这个软件。

从官网下载。

Anki

一个卡片式的记忆记忆软件。网上的教程很多，我也有计划写一个新的 blog。

sudo apt install anki

娱乐

网易云音乐

网易云音乐官方提供.deb格式的安装包。
下载

Spotify

snap install spotify

VLC 视频播放器

sudo snap install vlc

某些情况下安装软件时并不能走系统代理。

sudo apt install proxychains

qBittorrent

p2p下载工具。

sudo apt install qbittorrent

telegram

sudo snap install telegram-desktop

extract

一个解压工具，不需要记忆那些奇奇怪怪的解压命令。
sudo apt install extract

syncthing

去中心化的同步工具。

官网有详细的教程。

Send Any Where

官网。

方便在设备之间传输文件。

Calibre

一个电子书管理器。
sudo apt install calibre

Zathura

一个pdf阅读器，支持vim的操作方式。

使用 vim 写 $\LaTeX$时的好帮手。

sudo apt install zathura
