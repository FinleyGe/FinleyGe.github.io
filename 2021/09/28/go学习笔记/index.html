Go语言概述

Golang 是 Google 开发的编程语言：
特性：

  • 静态强类型
  • 编译型
  • 并发型
  • 具有垃圾回收功能

Go语言语法的强制性

  1. Go 语言没有行分割符（也即没有分号）
  2. 前花括号不能单独成行

Go 语言结构

六个结构：

  1. 声明包名, 每个 Go 应用程序都需要一个名为 main 的包
  2. 引入包
  3. 函数声明
  4. 注释
  5. 变量
  6. 语句
