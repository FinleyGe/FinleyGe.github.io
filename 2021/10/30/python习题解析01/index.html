2题

None 表示 空

解释
  • None 是一个特殊的对象，表示空。通过type函数可以发现，None是一个NoneType类型。

    对象 (Object)对于Python而言，万物皆对象。一个字符串，一个变量，一个函数，都是对象。对象是构成一个Python程序的基本元素。

类型 (Type) 每个对象都有一个类型。所以请牢记：一个数据，不只是它的值，还有它的类型。例如：”1”和1是不同的，前者是str类型，后者是int类型。

  • None 和 “” 以及 [] 是不同的。

3题

结果分别是 True 和 False

解释

isinstance()函数需要传入两个参数
第一个参数是一个值
第二个参数是一个类型

函数 例如 isinstance() 就是一个内置函数，coder也可以自定义函数。函数由三大部分组成，即

1. 参数列表
2. 函数体
3. 返回值

以isinstance()为例，它的参数列表即为一个值和一个类型，它的函数体是预先定义好的，而返回值是一个布尔值。

import语句为导入库（包）。一个库，是python官方或第三方写好的一个文件。用户通过导入，可以直接使用库中的函数（类、对象等）。

and运算符被称为逻辑与运算符。对于x and y，如果x是False 那么返回x的值，否则返回y的值。

18题

a = 1.0
b = 2.0
sita = 10.0

输出['0','1', '2', '3', '4']

解释

map函数，即映射。需要传入两个参数，第一个参数是一个函数，第二个参数是一个可迭代对象。该函数将把第二个参数中的每一个元素作为参数，调用第一个参数传入的函数中，将该函数的返回值返回（实际上，map函数返回的也是一个map对象。）

例如第一问，float函数是显式转换为float类型。可迭代对象是一个列表。map函数将列表中的每一项用float函数调用。返回的是一个map对象，再隐式转换为元组，给三个变量赋值。
第二问类似。

19题

解释

导入了math包。
sqrt函数开平方，cos函数算余弦，radians将角度转换为弧度。
input函数传入一个参数（也可以不传，这个参数只是作为提示）返回的是一个字符串。必须对字符串处理为一定的类型才能使用。

split是字符串的一个方法，传入一个字符串参数。该方法将字符串以参数分割，返回一个列表。该参数默认为空格。

方法 类似于函数。不过这个函数是某一个对象的函数，一般称之为方法。
