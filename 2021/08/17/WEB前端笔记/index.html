1. HTML

概述

HTML： 超文本标记语言
使浏览器表达一定的内容的语言。

HTML由标签组成

标签

标签由尖括号括住。

标签的分类

  1. 块级标签：占一整行的标签，例如<div>标签
  2. 内联标签：有多少内容占多少空间，例如<span>标签

另一种分类：

  1. 自闭合标签：例如<input>标签
  2. 一般标签：有开始和结束,例如<div></div>

文件树

一般说来，一个HTML文件：

head 标签

meta 标签

  1. 设置页面编码
  2. 设置刷新跳转
  3. 设置关键词
  4. 描述
  1. 适配IE

title 标签

link 标签

  1. 引入CSS
  1. 设置图标

style 标签

body 标签

一些符号

p 标签

表示段落

a 标签

  1. 链接到别的网页
  2. 锚，连接到本文的某处。

H 系列标签

input 标签

  1. 文本，密码，电邮。
  1. 多行文本
  1. 单选框
  1. 复选框
  1. 下拉框 若要多选

size 表示页面上一次显示几个

选项分类

  1. 上传文件

  2. 提交

  3. 重置

  4. 普通按钮

form 标签

表单，在这个标签内的内容会被提交。

如果要上传文件，必须写enctype=。如果不写，上传的只是文件名

label 标签

一般没用
特殊作用：

点击label标签可以自动移动光标

无序列表

有序列表

标题-内容

table 表格

其他标签

  1. 标题框
  1. 嵌入网页
  1. img 显示图片

src –> 图片位置
alt –> 图片提示

2. CSS

CSS就是层叠样式表
类似PS的图层

CSS可以存放在三个位置：

  • 文件中
  • 头标签中
  • 标签属性

上述三个位置由上到下优先级增加

选择器

三种选择器：

标签选择器

id选择器

class选择器

选择器的使用：

  1. position 位置

absolute 绝对位置固定
relative 相对父标签固定，这个属性应该给父标签加
fixed 相对于用户窗口固定
